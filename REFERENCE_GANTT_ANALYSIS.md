### Best Practices Identified ✅

1. **Hierarchical Collapsible Structure**
   - Courses organized under logical categories (Class & Milestones, Job Interview, Research)
   - Allows complex information without visual clutter
   - Users can expand/collapse sections for detail

2. **Explicit Dependency Arrows**  
   - Visual dependency lines showing task relationships
   - Shows prerequisite flow (e.g., TECH 60900 "Prelim" → TECH 69900 with arrows)
   - Multi-stage tracking: "Professor Application 1 → Professor Application 2"

3. **Sequential Milestone Markers**
   - "Commencement" shown as endpoint
   - Research activities clearly categorized (Ph.D. Thesis, RA Projects)
   - Allows viewing both academic path and career path (Job Interview section)

4. **Mixed Timeline View**
   - Spans 2026-2029 with year-level granularity
   - Month markers visible for sub-semester detail
   - Quarterly/monthly breakdown on timeline header

### Critical Gaps for Rubric Compliance ❌

1. **NO explicit credit hours shown**
   - Rubric requires "Coursework mapped with credit hours"
   - No "3cr", "1cr" labels visible anywhere

2. **NO visible semester breakdown**
   - Courses listed without semester assignment
   - No clear "Fall 2026, Spring 2027" labeling

3. **Preliminary exam timing unclear**
   - Shows "Prelim" as milestone but timing relative to last course is ambiguous
   - No clear verification that Prelim occurs in final coursework semester

4. **Post-defense requirements missing**
   - No dissertation writing phase detail
   - No defense/graduation progression clarity
   - No administrative post-defense steps

5. **Committee evolution not visible**
   - No indication of when committee size changes
   - No committee member designations

### Unique Elements Worth Copying 🎯

- **Dependency arrows between tasks** - visual clarity of prerequisite relationships
- **Nested course structure** - hierarchical organization prevents wall-of-text effect
- **Career track integration** - Job Interview section shows parallel professional development
- **Multi-year span visualization** - shows full program scope at once

   - Shows logical progression visually

4. **Semester-Level Granularity**
   - Timeline spans Spring 2024 - Fall 2027
   - Semester-level task bars (appropriately coarse for high-level view)

1. **NO credit hours shown**
   - Past courses: CGT 512, 637, 697, 532, 525 - no credits
   - Current/planned: no credit information
   - Rubric explicitly requires credit hour mapping

2. **NO specific semester assignment**
   - Courses shown but which semesters they occupy is unclear
   - No semester number or academic calendar detail

3. **Unclear Prelim timing**
   - Marked as "Completed" but timing relative to coursework ambiguous
   - No verification of Prelim placement rubric (same semester as final course)

4. **Missing course prerequisites**
   - No indication of course sequence or dependencies
   - No connection between courses shown

5. **No administrative requirements**
   - No committee information

   - No IRB, RCR, or other mandatory requirements

6. **Vague milestone timing**
   - "Dissertation writing Plan" - but when?

- **Status tracking approach** - "Completed," "In Progress," "Plan" is useful for ongoing students
- **Milestone list consolidation** - single-page milestone view is scannable
